Git & Github

There is a difference between "git" and "github".

Upload everything to Github. Shows progress and helps us get in habit.

3 Commands:

  1. Git add:
    • Adds untracked file(s) to an upcoming commit
    • You do not have to add 1:1. Just what you want to add for the next commit.
  2. Git commit
    • Will stage the changes that were made, and added, to be commited or logged.
  3. Git push
    • Commits are only local. You have to "push" commits, in order to, have them on a different system.

Git Log will show a history of all commits for a project. It, in theory, should show you the history/building blocks of the project.

Commit often!You can't over-commit.
